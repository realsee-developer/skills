# 行为准则

[English](CODE_OF_CONDUCT.md) | 简体中文

本项目期待尊重且建设性的协作。

## 期望行为

- 使用友好、专业的语言。
- 将反馈聚焦在工作本身及其影响上。
- 尊重不同经验水平和工作环境。
- 让讨论保持与仓库相关。

## 不可接受行为

- 骚扰、威胁、侮辱或歧视性语言。
- 未经许可发布私人信息。
- 分享 secrets、凭证、私有 URL 或账号标识。
- 反复干扰技术讨论。

## 执行

维护者可以编辑、隐藏或删除违反本准则的评论、issues、pull requests 或其他贡献。对于反复或严重违规行为，维护者也可以限制参与。

行为相关问题可通过 `SECURITY.md` 中列出的仓库安全或维护者联系路径报告。
